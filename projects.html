🤖 Agentic AI Track · 11 Labs + 1 Capstone

Production-Grade Agentic AI Projects

Each project comes straight from the labs — the kind of autonomous-ops challenges teams hit in real production systems. You'll have built and hardened them before you're asked about them.

🤖 Agentic Foundations & Tooling 3 Projects
01 Prometheus Anomaly Explainer — A ReAct agent that queries live Prometheus metrics and explains anomalies in plain language.
02 kubectl Copilot — A tool-calling agent that interprets kubectl output and proposes concrete remediation steps.
03 Safe Actuation Agent — Extends the kubectl copilot to execute approved actions against a staging cluster behind a confirmation gate.
🚀 Agentic CI/CD 2 Projects
04 LLM Pull-Request Reviewer — An LLM reviewer wired into GitHub Actions that comments on diffs, flags risks and suggests fixes.
05 Deployment Risk Scorer & Pager — A deployment agent that parses error logs, scores rollback risk and pages on-call with a reasoned summary.
📊 Observability & Incident Response 2 Projects
06 RAG-over-Logs Pipeline — A retrieval pipeline that embeds and queries a synthetic log corpus for fast, grounded answers.
07 Incident-Response Agent — Correlates alerts, retrieves the right runbook sections and drafts a Slack incident summary with a timeline.
🏗️ Infrastructure as Code 1 Project
08 Terraform Drift Remediator — Detects drift from Terraform state, generates a remediation plan, asks for Slack approval and applies only after confirmation.
🛡️ Security & Guardrails 1 Project
09 Red-Team & Harden an Ops Agent — Find prompt-injection vectors in an ops agent, add input/output guardrails and structured audit logging, then verify the hardened build in tests.
🕸️ Multi-Agent & MCP 2 Projects
10 Multi-Agent Ops Orchestrator — An orchestrator that delegates to specialized subagents for cost analysis, security scanning and deployment health.
11 MCP Server for Your Ops Agents — Exposes the multi-agent system via a Model Context Protocol server so it's usable directly from your IDE.
🧩 Capstone 1 Project
12 Autonomous Ops Platform — Design, integrate, harden and demo a working autonomous ops platform - CI/CD hooks, observability feeds, IaC tooling and Slack/PagerDuty, tested under chaos and adversarial prompts.
